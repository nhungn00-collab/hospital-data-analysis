# Hospital Data Analysis 

This project analyzes hospital performance to evaluate treatment effectiveness and patient satisfaction.

## Overview
- Dataset: 984 patient records
- Tools: SQL, Power BI
- Focus: Treatment success, LOS, cost, satisfaction, readmission

## Key Insights
- Longer hospital stays reduce patient satisfaction
- Higher cost does not guarantee better outcomes
- Surgery and combined treatments show higher success rates
- Readmission is linked to lower satisfaction
