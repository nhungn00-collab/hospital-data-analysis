Project Overview

This project analyzes hospital performance using SQL and Power BI to evaluate treatment effectiveness and patient satisfaction.

Objectives

  • Evaluate treatment success rate
  • Analyze patient satisfaction
  • Identify key drivers affecting outcomes
  • Provide data-driven recommendations

Dataset

  • 984 cleaned patient records
  • Handled missing values & duplicates using SQL
  • Visualized insights using Power BI

Key Insights

  • Treatment effectiveness varies across conditions
  • Surgery and combined treatments show higher success rates
  • Longer hospital stays reduce patient satisfaction
  • Higher cost does not guarantee better outcomes
  • Readmission is linked to lower satisfaction

Impact: Reducing length of stay (LOS) can improve patient satisfaction and optimize hospital costs.

Recommendations

  • Optimize treatment protocols
  • Reduce unnecessary LOS
  • Improve post-treatment care
  • Digitize hospital processes
